**Handloom and Craft Heritage Expo Portal**

This project is a simple Handloom and Craft Heritage Expo portal developed using HTML.
It is designed to showcase traditional handloom products and cultural crafts while providing information about the expo.

**Features**

Display of handloom and craft images

Informational video about heritage crafts

Audio element for cultural ambience

Visitor registration form

**Technologies Used**

HTML5

Basic multimedia elements (image, video, audio)

**How to Run**

Open the index.html file in any web browser.

**Purpose**

This project is created for academic learning to understand web page structure, multimedia integration, and form handling.
